###### Processus d'installation (DOCKER)
                        
          1. Installer docker desktop https://docs.docker.com/desktop/ et le lancer
          2. Visual Studio Code https://code.visualstudio.com/download/ et le lancer
          3. Entrer dans le dossier clone local de github avec VS Code
          4. Entrer docker compose up --build dans le terminal du VS Code pour lancer le site
